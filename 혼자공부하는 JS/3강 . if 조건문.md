# 조건문
- Bolean 자료형을 활용한 true / false 활용문
- 비교연산자 && 일치연산자를 활용하여 조건문을 만들고, 분기를 정하는 방법

## if 문 기본 문법 구조

```JavaScript
if ( true || false ) {
	true || false 조건에 부합할 때 작동하는 CODE
} else {
	(if 부분이 성립하지 않을 때) 실행되는 CODE => 조건은 따로 넣지 않는다.
}
```

```JavaScript
if (String(originpassword) === String(password)) {
```
